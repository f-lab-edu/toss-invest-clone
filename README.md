# 토스증권 웹 클론코딩

토스증권 웹의 UI/UX와 주요 기능을 분석하고 재구현한 클론코딩 프로젝트입니다.
실제 서비스와 유사한 화면과 인터랙션을 구현하며 **좋은 설계와 테스트 코드 작성**을 학습하는 데 중점을 두었습니다.

<br>

## 🛠 기술 스택

- **Frontend:** React, Vite
- **Test:** Vitest
- **Language:** TypeScript, JavaScript (ES6+)
